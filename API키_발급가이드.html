YouTube API 키 발급하기

유튜브 레퍼런스 수집기를 사용하려면 API 키가 필요합니다

⏱ 약 5~10분 💰 완전 무료 🔁 한 번만 발급

0 시작하기 전에

이 가이드는 Min(남편)이 본인 Google 계정으로 키를 발급한 뒤, 두 컴퓨터에서 같은 키를 사용하는 방식입니다.

준비물

  • Google 계정 (Gmail 있으면 그걸로 OK)
  • 크롬 또는 엣지 브라우저
💡 Google Cloud Console이 처음이라도 괜찮아요 무료 등급으로 충분하고, 결제 정보 등록 안 해도 됩니다. API 키만 발급받으면 끝이에요.

1 Google Cloud Console 접속

  1. 아래 주소로 이동합니다.
    https://console.cloud.google.com
  2. Google 계정으로 로그인합니다.
  3. 처음이라면 약관 동의 화면이 뜹니다. 국가 선택 후 동의 체크하고 진행.

2 새 프로젝트 만들기

  1. 화면 좌측 상단에 Google Cloud 로고 옆 프로젝트 선택 드롭다운이 있어요. 클릭합니다.
    • 처음이면 "프로젝트 선택" 또는 "프로젝트가 없습니다"라고 표시될 수 있어요.
  2. 오른쪽 위 새 프로젝트 버튼을 클릭합니다.
  3. 프로젝트 이름을 입력합니다. 예: youtube-reference
  4. 만들기 버튼을 누릅니다.
  5. 잠시 후(보통 5초 이내) 알림창에 "프로젝트가 생성되었습니다"가 뜨면 프로젝트 선택을 눌러 그 프로젝트로 전환합니다.
💡 프로젝트가 잘 선택됐는지 확인 화면 위쪽 가운데에 방금 만든 프로젝트 이름이 표시되어 있어야 합니다. 다른 이름이면 드롭다운 다시 눌러서 선택하세요.

3 YouTube Data API v3 사용 설정

  1. 화면 왼쪽 상단의 햄버거 메뉴(≡)를 클릭합니다.
  2. API 및 서비스 → 라이브러리 클릭.
  3. 검색창에 YouTube Data API v3를 입력합니다.
  4. 결과에서 빨간색 유튜브 아이콘이 있는 YouTube Data API v3를 클릭.
  5. 파란색 사용 또는 사용 설정 버튼을 누릅니다.
  6. 잠시 로딩이 끝나면 API가 활성화됩니다.

4 API 키 생성

  1. 다시 햄버거 메뉴(≡) → API 및 서비스 → 사용자 인증 정보 클릭.
  2. 상단의 + 사용자 인증 정보 만들기 클릭.
  3. 드롭다운에서 API 키 선택.
  4. "API 키가 생성되었습니다" 창이 뜨면서 키가 표시됩니다.
  5. 키 옆 복사 버튼을 눌러 어딘가에 메모해두세요.
    • 키는 보통 AIzaSy... 로 시작하는 39자 문자열입니다.
    • 메모장이나 카톡 "나에게 보내기"에 잠깐 저장해두면 두 컴퓨터에서 쓰기 편해요.

5 키 보안 설정 (강력 권장)

키를 그대로 두면 누가 알아낼 경우 다른 용도로 쓰일 수 있어요. YouTube Data API만 사용할 수 있게 제한해두는 게 좋습니다.

  1. 방금 만든 키 옆의 편집 아이콘(연필 모양) 또는 키 이름을 클릭합니다.
  2. 스크롤을 내려 "API 제한사항" 섹션을 찾습니다.
  3. 키 제한 라디오 버튼을 선택합니다.
  4. 드롭다운에서 YouTube Data API v3를 체크합니다.
  5. 맨 아래 저장 버튼을 누릅니다.
⚠️ 키는 절대 외부에 공개하지 마세요 GitHub, 블로그, SNS 등에 키를 올리면 누가 가져가서 할당량을 다 써버릴 수 있습니다. 본인과 아내분만 알고 있도록 관리하세요.

6 두 컴퓨터에 키 입력하기

이제 발급받은 키를 도구에 넣어주면 끝입니다.

Min님 컴퓨터

  1. 폴더의 index.html 더블클릭.
  2. API 키 입력창이 뜨면 복사해둔 키를 붙여넣고 저장.
  3. 검색 한 번 해보고 잘 되면 OK.

아내분 컴퓨터

  1. 폴더를 옮긴 뒤 아내분 컴퓨터에서 index.html을 한 번 엽니다. (Min님이 같이 해주세요.)
  2. 같은 키를 입력해서 저장.
  3. 한 번 입력해두면 브라우저가 기억하기 때문에 이후엔 아내분이 그냥 더블클릭만 하시면 됩니다.
💡 키는 컴퓨터에만 저장돼요 입력한 키는 그 브라우저의 localStorage에 저장됩니다. 외부로 전송되지 않으니 안전합니다. 단, 브라우저 데이터를 모두 삭제하면 키도 사라져서 다시 입력해야 해요.

7 할당량 알아두기

Google이 정한 무료 사용량을 알면 도구를 더 잘 활용할 수 있습니다.

항목 수치
하루 무료 할당량 10,000 units
검색 1회 비용 약 102 units
하루 가능한 검색 횟수 약 98회
리셋 시각 (한국 기준) 매일 오후 5시 (PT 자정)

Min님과 아내분이 같이 쓰는 구조이므로, 두 분 합쳐서 하루 98회까지 검색 가능합니다. 보통 레퍼런스 수집은 하루 10~20회 정도라 충분합니다.

할당량 확인하는 법

  1. Google Cloud Console에서 햄버거 메뉴(≡) → API 및 서비스 → 사용 설정된 API 및 서비스
  2. YouTube Data API v3 클릭
  3. 할당량 및 시스템 한도 탭에서 오늘 사용량 확인 가능

8 문제 생겼을 때

"이 API 키는 권한이 없습니다" 에러

5단계에서 키 제한을 설정한 직후엔 적용까지 1~2분 정도 걸릴 수 있어요. 잠깐 기다렸다가 다시 시도해보세요.

"API key not valid" 에러

키를 복사할 때 앞뒤에 공백이 섞였을 수 있어요. 다시 복사해서 정확히 붙여넣어주세요. AIzaSy로 시작해야 정상입니다.

"quotaExceeded" 에러

하루 할당량 10,000을 다 썼다는 뜻입니다. 한국 시각 오후 5시에 자동으로 리셋돼요.

키를 잃어버렸어요

Google Cloud Console → 사용자 인증 정보 메뉴에서 언제든 다시 확인하거나 새로 발급 가능합니다. 새로 발급받은 경우 두 컴퓨터에 다시 입력해주세요.
